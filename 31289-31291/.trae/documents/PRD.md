# 高尔夫球模拟小游戏 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
开发一款基于Web的高尔夫球模拟小游戏，提供沉浸式的高尔夫运动体验。玩家可以在18洞标准球场上体验高尔夫运动的乐趣，通过拖拽操作控制击球方向和力度，感受真实的物理效果。

### 1.2 目标用户
- 休闲游戏爱好者
- 高尔夫运动爱好者
- 寻求轻松娱乐体验的网民

### 1.3 核心价值
- 简单易上手的操作方式
- 真实的物理模拟效果
- 18洞标准球场体验
- 完整的计分系统

## 2. 产品特性

### 2.1 核心功能

#### 2.1.1 游戏画面
- **俯视视角**：采用2D俯视视角展示整个高尔夫球场
- **精美球场渲染**：包括草地、沙坑、水池、树木等元素
- **发球台**：每洞起始位置标识清晰
- **球洞**：目标位置，配有旗杆标识

#### 2.1.2 操作控制
- **拖拽击球**：玩家通过拖拽操作调整击球方向
- **力度控制**：拖拽距离决定击球力度
- **方向指示**：拖拽时显示击球方向和力度预览

#### 2.1.3 物理系统
- **抛物线飞行**：球飞行轨迹遵循抛物线运动规律
- **地形影响**：球落地后滚动受地形坡度影响
- **碰撞检测**：球与障碍物（树木、沙坑、水池）的交互
- **入洞判定**：球进入球洞的精准判定

#### 2.1.4 游戏流程
- **18洞球场**：标准18洞高尔夫球场
- **标准杆**：每洞设定标准杆数（3-5杆）
- **逐洞进行**：按顺序完成18洞
- **计分系统**：记录每洞杆数，统计总杆数

#### 2.1.5 计分板
- **实时显示**：当前洞数、标准杆、已用杆数
- **总杆统计**：18洞总杆数汇总
- **成绩对比**：显示与标准杆的差值（低于/高于标准杆）
- **成绩记录**：保存各洞成绩

### 2.2 用户界面

#### 2.2.1 主界面
- 游戏标题
- 开始游戏按钮
- 操作说明

#### 2.2.2 游戏界面
- 中央：高尔夫球场画布
- 顶部：当前洞信息、标准杆、已用杆数
- 底部：力度指示器、操作提示
- 侧边：计分板（各洞成绩）

#### 2.2.3 结算界面
- 18洞总成绩
- 各洞详细成绩
- 与标准杆对比
- 重新开始按钮

## 3. 交互设计

### 3.1 击球操作
1. 鼠标按下：开始瞄准
2. 鼠标拖动：调整方向和力度（显示预览线）
3. 鼠标释放：击球，球按设定的方向和力度飞出

### 3.2 游戏流程
1. 进入游戏 → 显示第1洞
2. 玩家击球 → 球飞行 → 落地滚动 → 停止
3. 重复击球直到球入洞
4. 显示该洞成绩 → 进入下一洞
5. 完成18洞 → 显示总成绩

### 3.3 视觉反馈
- 拖拽时显示力度条和方向线
- 击球时有动画效果
- 入洞时有庆祝动画
- 计分板实时更新

## 4. 技术要求

### 4.1 技术栈
- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript**：游戏逻辑
- **Canvas API**：游戏渲染

### 4.2 性能要求
- 60fps流畅动画
- 响应式设计，适配不同屏幕尺寸
- 快速加载，无需额外依赖

### 4.3 兼容性
- 支持主流现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持桌面端鼠标操作

## 5. 设计风格

### 5.1 视觉风格
- **清新自然**：以绿色为主色调，模拟真实高尔夫球场
- **简约现代**：UI元素简洁大方
- **生动活泼**：流畅的动画效果

### 5.2 色彩方案
- 主色：草地绿 (#228B22)
- 辅色：天空蓝 (#87CEEB)、沙黄色 (#F4A460)
- 强调色：红色旗杆、白色高尔夫球

### 5.3 字体
- 标题：粗体无衬线字体
- 正文：清晰易读的无衬线字体
